Intelligent Communication and Information Systems

Working on cyber-emulation

Our tool provides a virtual infrastructure for network emulation. We allow users to configure testbeds where they can run experiments. Having a virtualized network environment allows scientists to quickly deploy, test, modify, and tear down network configurations. With Virtual Distributed Ethernet (VDE), testbeds can scale horizontally as experiments grow.

Technologies: Java, MySQL, VirtualBox, VDE, JavaScript, Ember.js

Worked on electronic warfare simulation

Our tool—a game for the Spring RTS engine—provided proof-of-concept for an electronic warfare simulation. By redirecting commands out to an emulated wireless network, we were able to simulate radio limitations in the game.

Related publication: A Gaming Environment for Resilient Network Design and Adversarial Co-Evolution Modeling

Technologies: Lua, C++
